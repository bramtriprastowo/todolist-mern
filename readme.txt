Aplikasi TodoList

--- Backend
Untuk back end, menggunakan framework Mongoose. Pada aplikasi ini terdapat API yang sudah didokumentasikan melalui swagger, dapat diakses melalui link '/api-docs'. 

--- Front end
Untuk front end nya, dibangun menggunakan Vite, dan menggunakan framework Bootstrap. Untuk melakukan start dapat menggunakan perintah 'npm run start'.
